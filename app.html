Help
  • When clicking the "Anonymize" button, the text is saved in a simple database (holding only this one essay, for demo purposes), and then Svala is opened
  • Svala will request the same essay from the database
  • Subsequent edits in Svala will be saved to the database
  • When going back from Svala, this page will load the (updated) essay and display the target version
  • The "Reset" button will delete the essay from the database, and you can try the workflow again
